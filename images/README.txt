# 在这里放置 12 张贴纸图片

文件命名规则:
  0.png   ←  0 点 / 12 点 发送这张
  1.png   ←  1 点 / 13 点
  2.png   ←  2 点 / 14 点
  3.png   ←  3 点 / 15 点
  4.png   ←  4 点 / 16 点
  5.png   ←  5 点 / 17 点
  6.png   ←  6 点 / 18 点
  7.png   ←  7 点 / 19 点
  8.png   ←  8 点 / 20 点
  9.png   ←  9 点 / 21 点
  10.png  ←  10 点 / 22 点
  11.png  ←  11 点 / 23 点

也支持 .jpg / .jpeg / .gif / .webp，
按 _conf_schema.json 中 image_ext_priority 顺序匹配。

如果想 24 小时不同图，开启 use_24h_mode，
然后放 0.png ~ 23.png 共 24 张。

放完图片后可以删除本说明文件。
